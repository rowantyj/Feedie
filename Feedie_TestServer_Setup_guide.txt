
Setting up Test Server

Step 1: Download XAMPP Server
Step 2: go to "XAMPP/htdocs/" , copy & paste Folder "feedie" into it.
Step 3: type "localhost" on browser.
Step 4: go to phpMyAdmin, create a database called "Feedie_Test"
Step 5: Select "Import", import file "Feedie_Test.sql".


Within Android Code:

Step 1: Go to LoginPage.java & RegisterPage.java
Step 2: find LOGIN_URL
Step 3: Replace the http link <your ip address>:<port> (default is 80)
